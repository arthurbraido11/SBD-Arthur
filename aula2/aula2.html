Começando com CSS

Vimos que o HTML é uma linguagem que permite inserir marcações para representar tipos de conteudos e fazer link com os outro arquivos(outras páginas). É chamado hipertexto.

O CSS por sua vez, tem a responsabilidade de formatar uma página HTML, por exemplo, alterando cores de texto, de fundo, tipos e de tamanhos de fontes (letras) etc.
